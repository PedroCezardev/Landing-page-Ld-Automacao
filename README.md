# LD Automação - Landing Page

Esta é a landing page para a **LD Automação**, uma empresa especializada em reparos de celulares, tablets e outros dispositivos eletrônicos.

## Sobre o Projeto

O objetivo deste projeto é fornecer uma presença online moderna e profissional para a LD Automação, permitindo que os clientes conheçam os serviços oferecidos, entrem em contato e encontrem a localização da empresa.

## Tecnologias Utilizadas

*   **Next.js**: Framework React para renderização do lado do servidor e geração de sites estáticos.
*   **React**: Biblioteca para construção de interfaces de usuário.
*   **Tailwind CSS**: Framework de CSS para estilização rápida e responsiva.
*   **TypeScript**: Superset de JavaScript que adiciona tipagem estática.
*   **Vercel**: Plataforma para deploy e hospedagem do projeto.

## Como Executar o Projeto

Siga os passos abaixo para executar o projeto em seu ambiente de desenvolvimento local:

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/seu-usuario/ld-automacao.git
    cd ld-automacao
    ```

2.  **Instale as dependências:**
    ```bash
    npm install
    ```

3.  **Execute o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```

4.  **Abra no navegador:**
    Abra [http://localhost:3000](http://localhost:3000) em seu navegador para ver o resultado.
